# Corpus de référence — CarryIT

## Rôle
Ce fichier définit les références que le futur skill doit utiliser pour analyser, concevoir et critiquer l’UX/UI de CarryIT.

## Sources prioritaires
1. WCAG 2.2
2. Heuristiques de Nielsen
3. Principes UI web
4. Patterns UX pour dashboard, formulaires, feedback et tâches
5. Modèles comportementaux compatibles avec la vérité produit

## Règle de priorité
En cas de conflit :
accessibilité > utilisabilité > cohérence UI > modèles comportementaux

## Exclusions
- modèles de rétention addictive
- gamification manipulatoire
- récompenses variables artificielles
