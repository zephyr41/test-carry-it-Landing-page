# Texte + Sous-texte — Règles copy

## Principe central
Tout copy a deux couches : le texte (les mots) et le sous-texte (l'émotion que les mots activent). Le sous-texte est limité — 4 émotions universelles. Le texte est infini — mille façons d'activer la même émotion. Le copy puissant aligne les deux parfaitement.

---

## Règles applicables

### Les 4 émotions universelles (FESN)
Tout achat, toute décision est motivée par une (ou plusieurs) de ces 4 émotions :

**1. Facile (Easy)** — "Je veux que ce soit simple. Je ne veux pas galérer."
Tout ce qui réduit la friction perçue active cette émotion.

**2. Sûr (Safe)** — "Je veux être sécurisé. Je ne veux pas prendre de risque inutile."
Besoin de certitude, de validation, de ne pas avoir l'air idiot si ça ne marche pas.

**3. Nouveau (New)** — "Je veux quelque chose que je n'ai pas encore essayé."
L'attrait du neuf, de la méthode différente, de l'angle qu'on n'avait pas vu.

**4. Big / FOMO** — "Je veux que ça change vraiment les choses. Je ne veux pas passer à côté."
Désir de résultats significatifs + peur de manquer quelque chose d'important.

### Sous-texte vs Texte
- Le **sous-texte** (l'émotion) est fixe — tu choisis laquelle activer.
- Le **texte** (les mots) est infini — mille façons d'activer la même émotion.

Exemple — sous-texte "Facile + Sûr" :
- Version banale : "C'est simple à utiliser."
- Version puissante : "Setup en 10 minutes. Après ça, le système fait le calcul — pas toi."

Chaque mot véhicule une émotion ou un ressenti. Deux phrases qui "disent la même chose" peuvent activer des émotions très différentes.

Exemple classique :
- "Une vieille femme dans une maison." — neutre
- "Une petite vieille dans sa bicoque." — image précise, émotion différente

### Règle d'écriture : 110% puis réduire
Écrire en poussant les émotions à fond (110%). Puis, à l'édition, réduire l'intensité pour trouver le niveau qui mettra l'audience à l'aise.

Mieux vaut partir trop fort et calibrer, que partir trop fade et ne jamais trouver le bon niveau.

### Les 6 questions d'optimisation du sous-texte
Quand tu relis un bloc de copy :

1. Parmi les 4 émotions, laquelle est la plus importante à activer ici, pour cette audience précise ?
2. Où / comment pousser encore plus l'émotion **Facile** ?
3. Où / comment pousser encore plus l'émotion **Nouveau** ?
4. Où / comment pousser encore plus l'émotion **Sûr** ?
5. Où / comment pousser encore plus l'émotion **Big / FOMO** ?
6. Dois-je monter l'intensité ou l'atténuer pour cette audience ?

### Processus complet
1. **Identifier les conditions spécifiques** — sans connaître l'audience, impossible de choisir le bon texte
2. **Choisir l'émotion cible** pour ce bloc
3. **Écrire plusieurs versions** du même message
4. **Se mettre dans la tête de l'audience** pour choisir la version qui résonne le plus

---

## Ce qui tue cette technique

- Activer plusieurs émotions contradictoires dans la même phrase
- Texte générique qui pourrait être écrit pour n'importe quelle audience
- Sous-texte non adapté à l'audience : pousser "Nouveau" pour une audience qui veut du "Sûr"
- Mots trop faibles pour l'émotion visée
- Intensité mal calibrée : trop fort = manipulateur / trop doux = rien ne se déclenche
